Project Statement



## Problem Statement
Users often need a quick and reliable way to convert an amount from one currency to another. Manual conversion requires checking exchange rates and performing calculations, which can be time-consuming and error-prone. This project provides a simple Python-based currency converter using predefined exchange rates for fast and accurate conversions.

## Scope of the Project
- Supports four major currencies: USD, INR, EUR, and GBP.
- Converts any user-input amount from one currency to another using a matrix of predefined exchange rates.
- Provides a console-based interaction suitable for beginner-friendly usage.
- Does *not* include real-time exchange rate updates or internet-based API integrations.

## Target Users
- Students learning Python and basic programming concepts.
- Users needing quick offline currency conversions.
- Beginners practicing NumPy and matrix operations.
- Individuals requiring a lightweight, terminal-based currency conversion tool.

## High-Level Features
- Display list of supported currencies.
- Accept user input for source and target currencies.
- Accept amount in source currency.
- Perform currency conversion using matrix multiplication.
- Display the converted amount clearly.
